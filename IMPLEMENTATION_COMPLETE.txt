📋 구현된 기능 상세
================================================================================

[로그인 페이지]
- 교사 로그인: ID(admin) / PW(1234)
- 학생 입장: 이름 + 6자리 접속 코드 입력 → Firestore 검증

[교사 대시보드]
- 과제 만들기:
  • 단원명, 지문, 난이도(Beginner/Intermediate/Advanced), 퀴즈 입력
  • 6자리 랜덤 코드 자동 생성
  • Firestore assignments 저장 (Document ID = 코드)
  • 생성 완료 메시지 + 풍선 효과

- 학습 결과 확인:
  • 모든 과제 코드 리스트 표시
  • Firestore submissions에서 제출 현황 조회
  • 학생 이름, 제출 시간, 점수 테이블
  • 오디오 재생 기능 (st.audio)

[학생 워크스페이스]
- 과제 로드: Firestore에서 과제 데이터 읽기
- 화면 구성:
  • 단원명, 난이도, 영어 지문 표시
  • streamlit-audiorecorder로 음성 녹음
  • "지문을 큰 소리로 읽고 녹음하세요" 안내문

- 제출 프로세스:
  • 오디오 바이트 → Firebase Storage 업로드
  • 경로: student_audio/{access_code}/{student_name}_timestamp.wav
  • Firestore submissions에 제출 정보 저장
  • 제출 완료 메시지 + 풍선 효과

[공통 기능]
- 사이드바: 현재 접속자 정보 (이름, 역할)
- 로그아웃: Session state 초기화 + st.rerun()
